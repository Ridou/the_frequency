### **Format Manuscript**

**Option A: DIY (Free-$250)**
- Use Vellum ($249 Mac only) - BEST results
- Use Atticus ($147 one-time) - Cross-platform
- Use Calibre (Free) - Basic but functional
- Use Draft2Digital (Free) - Auto-formatting

**Option B: Hire Formatter ($150-300)**
- Find on Reedsy or Upwork
- Provide edited manuscript
- Get back eBook + Print formatted files

**What You Need:**
1. **eBook file (.epub or .mobi)**
   - Linked table of contents
   - Proper chapter headings
   - No page numbers/headers
   - Copyright page
   - Back matter (Book 2 teaser, review request)

2. **Print file (PDF)**
   - 5"x8" trim size (standard)
   - Proper margins (use KDP template)
   - Page numbers
   - Chapter headings
   - Copyright page
   - Dedication (optional)
   - Back matter

**Time Required:** 8-12 hours DIY, 1 week if hiring  
**Cost:** $0-300
